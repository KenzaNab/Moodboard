# MoodBoard — Visual Inspiration Collector

Save images, links and notes in beautiful visual boards. Tag and search your collection.

> React · Node.js · Express · Multer · SQLite

## Features
- Create multiple boards
- Add items: image upload, links, text notes
- Tag items and search by tag or content
- Masonry grid layout
- Delete items (removes file from disk too)

## Quick start

```bash
cd backend && copy .env.example .env && npm install && npm run dev
cd frontend && npm install && npm start
```
